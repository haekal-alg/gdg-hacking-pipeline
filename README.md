# gdg-hacking-pipeline

Recon + SQL injection fuzzing scripts for a hands-on cybersecurity workshop.

For: https://gdg.community.dev/events/details/google-gdg-on-campus-uin-syarif-hidayatullah-jakarta-south-tangerang-indonesia-presents-hands-on-workshop-cyber-security-2526-building-your-first-automation-hacking-pipeline/

## Sources

- `payloads.txt`: 500 SQL injection auth-bypass payloads from SecLists' [`Fuzzing/login_bypass.txt`](https://github.com/danielmiessler/SecLists/blob/master/Fuzzing/login_bypass.txt).
- `wordlist.txt`: curated by hand for this workshop, not pulled from an external list.
